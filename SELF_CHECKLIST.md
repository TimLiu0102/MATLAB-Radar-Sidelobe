# 联合优化结果自检清单（LFM 预补偿）

> 目标：快速判断“结果是否可信”，并定位异常指标（尤其是 SPEC 恒为 1.0）。

## 1. 先做 60 秒快速体检

- [ ] **趋势是否正确**：
  - 无预补偿 PSLR 比理想明显变差（通常会变差）。
  - 有预补偿 PSLR 比无预补偿明显改善。
- [ ] **代价是否可接受**：
  - PAPR 有上升，但不应飙升到很夸张（例如 > 6~8 dB，视系统而定）。
- [ ] **分辨率是否守住**：
  - 主瓣宽度不应明显变宽（或仅轻微变宽）。
- [ ] **可疑项排查触发**：
  - SPEC（频谱恢复误差）若长期固定为 1.0000 或 0.0000，立即进入第 4 节深查。

## 2. 指标定义一致性检查（最容易出错）

- [ ] **PSLR 定义固定**：
  - 主瓣搜索区、旁瓣搜索区在三种工况中必须完全一致。
  - 是否使用 dB：`20*log10(|x|/max|x|)`。
- [ ] **ISLR 定义固定**：
  - 主瓣积分窗口一致。
  - 能量定义应为幅度平方后积分。
- [ ] **主瓣宽度定义固定**：
  - 采用 -3 dB 或首零点宽度时，标准要统一。
- [ ] **PAPR 定义固定**：
  - `10*log10(max(|s|^2)/mean(|s|^2))`。
- [ ] **SPEC 定义固定**：
  - 明确是 L2 相对误差、带内误差、还是加权误差。
  - 比较对象必须是“同一频率网格”下的目标与输出。

## 3. 归一化与对齐检查（结果看起来“很好/很怪”的常见根因）

- [ ] **统一能量归一化时机一致**：
  - 在进入指标计算前做，且三种工况流程一致。
- [ ] **FFT 长度一致**：
  - 统一 `Nfft`，避免频率分辨率不同导致误差不可比。
- [ ] **频率轴严格一致**：
  - 使用相同索引/掩码（band mask）。
- [ ] **时延与相位对齐**：
  - 相关峰是否对齐后再计算旁瓣和频谱误差。
- [ ] **dB 与线性域不混用**：
  - 误差计算在同一域中进行（建议线性域计算误差，dB 仅用于展示）。

## 4. SPEC = 1.0000 专项排查（按优先级）

- [ ] **打印中间量**：
  - `norm(T-S)`, `norm(T)`, `norm(S)` 是否异常（如分母为 0 或极小）。
- [ ] **检查公式是否“钉死”为 1**：
  - 例如误写成 `norm(T-S)/norm(T-S)` 或被二次归一化覆盖。
- [ ] **检查目标/输出是否取错变量**：
  - 有无把同一变量赋给了 `T` 与 `S`，或比较了错误阶段的频谱。
- [ ] **检查带内掩码**：
  - mask 是否全 0、全 1、或索引越界。
- [ ] **检查 NaN/Inf**：
  - 任何 NaN/Inf 进入后续流程都可能让最终打印值失真。
- [ ] **检查打印变量是否为最终值**：
  - 是否打印了初始化值或上一轮循环值。

## 5. 优化过程健康度检查

- [ ] **目标函数收敛曲线**：
  - `J(k)` 是否单调下降或总体下降。
- [ ] **参数是否卡边界**：
  - `a0/a1/a2/alpha_reg` 是否长期贴边。
- [ ] **多初值复现实验**：
  - 至少 5 组随机初值，统计均值与标准差。
- [ ] **扰动鲁棒性测试**：
  - 对系统模型参数小扰动，结果不应完全失稳。

## 6. 基线与对照实验（证明“优化确实有效”）

- [ ] **与固定窗对照**：
  - 与 Hamming/Hann/Taylor 统一口径比较 PSLR/ISLR/BW/PAPR/SPEC。
- [ ] **去掉某一项损失做消融**：
  - 例如去掉频谱项，仅保留旁瓣项，观察指标变化方向是否符合预期。
- [ ] **不同带宽/脉宽重复**：
  - 参数换挡后趋势应保持一致。

## 7. 建议的“通过标准”（可按工程要求调整）

- [ ] 相比“无预补偿”，PSLR 至少改善 **6 dB**。
- [ ] 主瓣宽度变化不超过 **10%**。
- [ ] PAPR 增量不超过 **3 dB**（或满足发射机上限）。
- [ ] SPEC 不再固定常数，且随优化权重变化呈可解释趋势。
